# 🍔 Digital Menu

A responsive and interactive digital menu built with pure HTML, CSS and JavaScript.

This project demonstrates dynamic rendering, array manipulation, DOM interaction and responsive layout techniques.

---

## 🚀 Features

- Dynamic product rendering
- Vegan items filtering
- Price sorting (ascending)
- 10% discount application
- Search by product name
- Total price calculation
- Currency formatting using Intl API

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Grid & Responsive Design)
- JavaScript (ES6+)

---

## 💡 Concepts Applied

- Array methods (`map`, `filter`, `reduce`, `sort`)
- DOM manipulation
- Event handling
- State management
- Conditional rendering
- Responsive UI layout
